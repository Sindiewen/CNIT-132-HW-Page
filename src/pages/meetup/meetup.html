Good Links for any Software Engineer!

  • Google - Everything is a search away! A perfect resource for any engineer
  • Stackoverflow - Questions and Answers board! If you have a question, check here, it might have been answered already!
  • Github - Version control is basically mandatory these days! Learn Git and easily collaborate on projects! As well as check out other projects!
  • Reddit - There's a subreddit for everything! Find a great community and learn about your topic of choice!
    Some of my favorite subreddits:
    • r/ProgrammerHumor - Great programming memes
    • r/softwaregore - watch programs just have an issue
    • r/badcode - learn by not following these examples
    • r/SoftwareEngineering - A general subreddit for the world of software
  • Twitter - Basically invaluable for any developer for connecting with anyone in the software industry
